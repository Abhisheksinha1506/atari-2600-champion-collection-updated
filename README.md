# Atari 2600 Champion Collection

The most curated, hand-tested, English-friendly ROMset for the Atari 2600 game console.

---

## About This Collection

The Atari 2600 was the first videogame console I grew up with, and I spent hours playing Combat, Pitfall, Pong, and more. You never forget your first.

The 2600 revolutionized the industry and Atari became the trusted household name for gaming in the late '70s and early '80s. This era marks the birth of many game companies that still exist today, and a lot of legal precedents regarding hardware, software, advertising, and creative ownership rights.

This collection represents a preservation effort to maintain a complete, high-quality archive of Atari 2600 games that are:
- **Hand-tested** for functionality
- **English-friendly** for accessibility
- **Curated** to remove duplicates and non-working dumps
- **Organized** for easy navigation and use

## Technical Specifications

### ROM Formats

This collection uses two primary ROM file formats:

- **`.a26`**: The standard file extension for Atari 2600 ROM files
- **`.bin`**: Binary format, also commonly used for Atari 2600 ROMs

Both formats are functionally identical and can be used interchangeably with most Atari 2600 emulators. The difference is primarily in file naming convention.

### File Size Distribution

Atari 2600 cartridges came in various sizes, reflecting the hardware limitations of the era:

| Size | Count | Description |
|------|-------|-------------|
| 2K | 59 files | Smallest cartridge size |
| 4K | 341 files | Most common size |
| 8K | 150 files | Enhanced games |
| 8.3K | 9 files | Special bank-switching |
| 12K | 4 files | Larger cartridges |
| 16K | 44 files | Advanced games |
| 25K | 3 files | Special formats |
| 32K | 1 file | Maximum standard size |
| 64K | 1 file | Extended format |

### Hardware Context

The Atari 2600 (originally released as the Atari VCS in 1977) had significant technical limitations:
- **RAM**: 128 bytes
- **ROM**: Typically 2K to 4K (later expanded to 32K with bank-switching)
- **Resolution**: 160×192 pixels (NTSC), 160×228 pixels (PAL)
- **Colors**: 128 possible colors (limited palette per line)
- **Sound**: 2-channel audio with simple waveforms

### Region Information

Games in this collection include region tags:
- **(NA)**: North American version
- **(EU)**: European version (PAL)
- **(BR)**: Brazilian version
- **(DE)**: German version
- **(Prototype)**: Unreleased prototype
- **(NTSC Conversion)**: Modified to work on NTSC systems

## What's Excluded

To maintain quality and avoid clutter, the following have been removed:

- Absolute duplicates (same game data, different names)

## Emulator Compatibility

### Recommended Emulator: Stella

**Tested Version**: Stella 3.9.3.3

Stella is the most accurate and feature-rich Atari 2600 emulator, actively maintained and compatible with all major platforms.

#### Setup Instructions

1. **Download Stella**: Visit [stella-emu.github.io](https://stella-emu.github.io/)
2. **Install**: Follow the installation guide for your operating system
3. **Load ROMs**: 
   - Open Stella
   - File → Open ROM
   - Navigate to the game folder
   - Select the `.a26` or `.bin` file
4. **Recommended Settings**:
   - Display: OpenGL mode for best performance
   - Audio: 44.1kHz or 48kHz
   - Controls: Configure your gamepad or keyboard

#### Other Compatible Emulators

- **MAME** (Multiple Arcade Machine Emulator) - Supports Atari 2600
- **RetroArch** with Stella core - Multi-system frontend
- **a2600** - Lightweight emulator for older systems
- **Javatari** - Browser-based emulator (JavaScript)

### Configuration Tips

- **NTSC vs PAL**: Most games in this collection are NTSC. If you experience speed issues, check your emulator's region setting
- **Controllers**: Use a gamepad that supports at least 2 buttons for authentic experience
- **Display**: Enable CRT simulation filters for period-accurate visuals
- **Save States**: Most emulators support save states for convenient gameplay

## Usage Instructions

### Loading Games

1. Choose your preferred emulator (Stella recommended)
2. Navigate to the appropriate letter folder
3. Select the game file (.a26 or .bin)
4. The game will load automatically

### Recommended Settings for Authentic Experience

- **Aspect Ratio**: 4:3
- **Filtering**: CRT scanline simulation
- **Audio**: Original mono sound
- **Controls**: Joystick emulation with one button

### Troubleshooting

**Game won't load**: Ensure you're using a compatible emulator and the file isn't corrupted

**Wrong colors/speed**: Check region settings (NTSC vs PAL) in your emulator

**No sound**: Verify audio settings in emulator and system volume

**Controls not working**: Reconfigure controller mapping in emulator settings
